# 兼容性

> 需要 PHP >= 7.4

# 安装

- step 1: `git clone git@github.com:Yi-Jams/Yi-SoUrl.git`
- step 2: 修改 ~/api/config.php , 填写您的mysql信息 以及网址等信息
- step 3: 导入 ~/mysql.sql 至您的数据库
- step 4: 配伪静态 (~/nginx.conf 或 ~/.htaccess) 目前仅有 nginx 与 Apache 的伪静态配置.
- step 5: 开始使用
